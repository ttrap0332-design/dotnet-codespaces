### Import Functions
```javascript
import {
  getFolkloreMappingByName,
  getRandomFolkloreMapping,
  searchFolkloreMappings,
  getAllFolkloreMappings,
  generateEnhancedMetadata,
  displayFolkloreMapping
} from './scripts/folkloreUtils.js';
```

### Get Specific Beast
```javascript
const kitsune = getFolkloreMappingByName("Kitsune");
// Returns complete mapping with all properties
```

### Search by Keyword
```javascript
const results = searchFolkloreMappings("speed");
// Returns array of matching folklore mappings
```

### Get Random Folklore
```javascript
const random = getRandomFolkloreMapping();
// Returns random beast/myth mapping
```
